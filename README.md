## 🏔️ 关于齿鸣幽境 — 新手必读

> 齿鸣幽境是一个以**结构探险 + 机械动力 + 养老种田**为核心玩法的 Minecraft 整合包服务器。  
> 你可以在这里深入各类神秘地下结构，搭建复杂的机械工厂，也可以安安静静地耕田养殖。  
> 无论你是哪种玩法偏好，都欢迎来这片幽境安家。

---

### 1. 下载整合包

整合包通过 **NeoForge** 加载，请按以下步骤获取：

1. 前往本仓库的 [**Releases 页面**](../../releases/latest)
2. 下载最新版本的整合包文件（`.zip` 格式）
3. 请确认你下载的版本号与当前服务端版本一致（见[更新记录](#-更新记录)）

> ⚠️ **注意**：请不要使用旧版整合包加入服务器，版本不匹配会导致无法连接。

---

### 2. 安装整合包

**前置要求：**
- Java **21** 或更高版本（[下载 Java](https://adoptium.net/)）
- 正版 Minecraft Java Edition（或使用支持离线登录的启动器，如 HMCL / PCL2）
- 内存建议：**至少 6 GB** 分配给游戏，8 GB 更佳

**安装步骤（以 HMCL / PCL2 为例）：**

1. 打开你的启动器，选择 **「导入整合包」**
2. 选择刚才下载的 `.zip` 文件
3. 等待启动器自动安装 NeoForge 及全部模组
4. 安装完成后，**不要修改任何模组文件**（如需添加自用模组请先联系管理员确认兼容性）

> 💡 **推荐启动器**：[PCL2](https://github.com/Hex-Dragon/PCL2) | [HMCL](https://hmcl.huangyuhui.net/)

---

### 3. 基本设置与加入服务器

**推荐游戏设置（降低卡顿）：**

| 设置项 | 推荐值 |
|--------|--------|
| 视距 | 8 ~ 10 |
| 模拟距离 | 6 ~ 8 |
| 图形 | 快速 |
| 流畅光效 | 开启 |
| 粒子效果 | 最少 |

**加入服务器：**

1. 启动游戏，进入 **多人游戏**
2. 点击 **添加服务器**
3. 服务器地址填写：

```
your.server.address.here
```

> 请将上方地址替换为实际服务器 IP。有疑问请联系管理员。

---

### 📜 服务器须知

1. **禁止任何形式的作弊、外挂、X-Ray**，违者永封。
2. **禁止恶意破坏他人建筑**，请尊重每一位玩家的劳动成果。
3. **禁止刷屏、辱骂、人身攻击**，保持友好的交流环境。
4. **大型机械装置请控制规模**，避免过度占用服务器性能（TPS 友好服务器，请不要刷怪塔超过 3 层）。
5. **未经管理员许可，禁止私自修改服务端相关配置**。
6. 遇到 Bug 请在 [Issues](../../issues) 中汇报，而不是自行利用。
7. 服务器最终解释权归管理团队所有。
