## 🏔️ 关于齿鸣幽境 — 新人必读

> ⚠️ 烦请务必细看新人必读这一部分，对了解整合包内容有很大的帮助

齿鸣幽境是一个基于 **NeoForge** 平台的 Minecraft 整合包服务器，无插件，在设计上平衡了冒险与科技元素，加入了大量增强氛围感与沉浸感的模组，并辅以各类性能优化。其中每个模组都经过配置调整，使游玩体验更加合理。

整合包没有固定的核心玩法——你可以深入各类神秘的地下结构，搭建复杂的机械工厂，也可以安安静静地耕田养殖，建造雄伟的建筑亦或是温馨精致的小家。无论你是哪种玩法偏好，都欢迎来这片幽境安家。

> 💡 **Tips**：整合包的设计以「平淡而有味」为核心，因此在模组选择上做了很多取舍。由于时间原因没有对每个模组进行深入测试，欢迎随时 [提出建议](../../issues)。

---

### 1. 如何安装客户端

1. **选择安装位置**：选择一个剩余空间充足的位置，**强烈建议安装在固态硬盘（SSD）上**。整合包内含 [LOD 模组（VOXY）](#-lod-模组--voxy)，随着游玩时间增长，LOD 地形缓存数据可能变得相当庞大，**建议预留至少 20 GB 以上的可用空间**。

2. **下载并解压客户端**：前往 [玩家交流群](#-社区与联系) 的群文件，下载最新版本的客户端压缩包，将其解压到刚才的位置。

3. **运行启动器**：打开解压目录中的启动器 （即 **PCL**）。首次启动时，PCL 会自动检测并下载补全游戏核心文件，这一过程可能耗时较长，请耐心等待。**下载完成后暂时不要启动游戏。**

4. **安装 Java 环境**：前往交流群群文件下载配套的 Java 环境（默认提供 **Zulu 25**，这是一个针对性能优化的轻量级 JDK 发行版）。下载完成后直接安装，建议保持默认安装路径不变。

---

### 2. 游戏启动与配置

> ⚠️ **登录要求**：必须在 PCL 中以**正版微软账号**登录后才能加入服务器。如尚未购买，请前往 [Microsoft 官方](https://www.minecraft.net/) 购买正版 Minecraft Java Edition。

完成客户端安装后，在正式启动游戏前，请按以下步骤在 PCL 中完成配置：

**① 基础启动设置**（PCL → 设置 → 启动选项）

- **进程优先级**：调整为「高」
- **游戏 Java**：点击「自动搜索」，选择刚才安装的 Zulu 25
- **高性能显卡**：勾选「使用高性能显卡」（笔记本用户尤其注意）
- **JVM 参数** → 内存管理：选择「尽量使用分代 ZGC」以获得更平稳的内存回收表现

**② 内存分配**

由于 LOD 模组对内存需求较高，建议按以下情况分配：

| 系统内存 | 建议分配方式 |
|----------|-------------|
| 32 GB 及以上 | 手动分配 **16 GB 或更多** |
| 16 GB 及以下| 勾选「自动配置」并开启「启动前进行内存优化」 |

> 💡 手动分配内存前，可前往 PCL → 更多 → 百宝箱 → 内存优化 进行手动释放。

**③ 其余设置**

PCL 的其他选项（除外观自定义外）不建议随意调整，保持默认即可。

---

### 3. 加入服务器

1. 确认以上配置完成后，在 PCL 中启动游戏
2. 服务器ip已内置：进入 **多人游戏** → **齿鸣幽境**

### 4. 游戏主要模组介绍

#### ① LOD 模组 · VOXY

LOD（Level of Detail）模组的作用是为你探索过的区域生成低精度地形缓存，让你能在游戏中看到极远处的地形轮廓，大幅提升世界的宏观观感与沉浸感。目前主流的 LOD 模组有两个：

**遥远的地平线（Distant Horizons，DH）** 是老牌选择，功能更全面。它可以在本地自主加载并生成区块的 LOD 模型，无需亲自跑图，且同时支持多人服务器和单人游戏。缺点是视觉效果不如 VOXY，性能开销也相对更高。

**VOXY** 是后起之秀，更轻量、视觉效果更佳，但相关功能仍在持续完善中，目前对多人服务器的支持还不够完善。本整合包选用 VOXY。

**在服务器中，VOXY 的工作方式如下：** 服务器会将玩家周围 **12 个区块**范围内的地形数据发送至客户端，VOXY 收到后会在本地以 **10 个线程**（可在游戏设置中调整）并行处理，为每个区块生成对应的 LOD 数据并缓存在本地。

> **❗ 重要**：由于 VOXY 对多人服务器的支持尚不完善，**在多人服务器中你只能看到自己亲身经过的地方的远景**，未探索区域不会有 LOD 显示。

如果你希望一进入服务器就能看到完整远景，可以前往 [QQ 交流群](#-社区与联系) 的群文件，下载由服主亲自跑图生成的 VOXY LOD 数据包，将其放置到以下目录后重启游戏即可：

> ⚠️ **注意**：VOXY 的缓存数据会随着探索范围的扩大持续增长，这也是为什么建议在安装时预留至少 20 GB 空间。

#### ② 主要新玩法

**机械动力 Create**

机械动力是本整合包的核心科技模组之一。它以「旋转力」(其实就是扭矩)为基础能源，玩家通过搭建各种齿轮、皮带、机械臂等结构来实现自动化生产流程。从简单的自动农场，到复杂的矿石精炼流水线，甚至是一列穿越大陆的蒸汽火车，都可以在 Create 的框架下实现。本整合包还额外加入了多个 Create 扩展模组，进一步丰富了玩法深度。

**农夫乐事 · 森罗物语**

农夫乐事（Farmer's Delight）为游戏带来了完整的烹饪系统，新增大量食材、锅具与料理配方，让「吃」这件事变得有趣且有收益。配合森罗物语的农业内容扩展，种田养殖不再只是资源收集的手段，而是可以独立成一条完整的生活线玩法。

---

#### ③ 冒险结构

整合包加入了多套结构模组，让世界中的地上与地下都充满值得探索的目标：

| 模组 | 简介 |
|------|------|
| **地牢崛起之时**（When Dungeons Arise） | 在世界各处生成体量庞大的空中城堡、地下宫殿等巨型建筑，高难度、高收益，是探险玩家的终极目标 |
| **地牢建筑统合**（IDAS） | Integrated系列，包含对村庄和要塞的改动，整合多套建筑风格的结构包，新增大量风格各异的地下城与地表建筑，与机械动力有紧密联动 |
| **地下城与酒馆**（Dungeons and Taverns） | 以中世纪风格为主，在世界中生成酒馆、哨塔、营地等小型但精心设计的结构。古城和末地堡垒采用DaT结构 |
| **塔**（Towers and Towers） | 专注于在各类地形上生成风格迥异的高塔建筑，内有挑战与宝藏 |
| **更好的试炼**（Not Enough Trials） | 对原版试炼密室进行大幅扩展与改良，带来更丰富的机关与奖励 |
| **YUNG 系列结构优化** | 对原版地牢、要塞、废弃矿道、神庙等原生结构进行全面重制，使其更具规模感与探索价值 |
| **Hopo 系列结构优化** | 主要针对水下遗迹和废弃矿道 |

### 5. 服务器须知

1. **禁止任何形式的作弊、外挂、X-Ray矿物透视**，违者永封。
2. **禁止恶意破坏他人建筑**，请尊重每一位玩家的劳动成果。
3. **禁止刷屏、辱骂、人身攻击**，保持友好的交流环境。
4. **大型机械装置请控制规模**，避免过度占用服务器性能。
5. **未经管理员许可，禁止私自修改服务端相关配置**。
6. 遇到 Bug 请在 [Issues](../../issues) 中汇报，而不是自行利用。
